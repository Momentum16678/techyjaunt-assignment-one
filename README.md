# TechyJaunt Assignment - Bunnies Array Exercise

A simple JavaScript exercise from the TechyJaunt Backend Development track.

## What it does

- Creates an array of six bunnies
- Adds Mario to the end of the array
- Removes Lucy from the array
- Adds Luigi to the beginning of the array

## How to run

```bash
node bunnies.js
```

## Tech

- JavaScript
- Node.js
